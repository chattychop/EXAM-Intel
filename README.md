# EXAM-Intel
AI-Powered Exam Prediction & Study Platform
Created by G. Badrinath | YouTube: ChattyChop

Quick Start

Open the file in Chrome/Edge/Firefox
Click ➕ to create your account (name + password)
Fill your profile — class, college, target exam
Get free API key at dashboard.cohere.com/api-keys
Paste it in the red box on the Predictor tab → Save → done


Features

⏱️ Live countdown timer to your exact exam (150+ exams)
🔮 Paper Predictor — upload PDFs, get predicted questions + topics
📖 Flip-book viewer — predictions open as a real question paper
🤖 AI Chat — knows your exam, syllabus, books, strategy
🔢 Scientific Calculator — same one the AI uses
📚 Exam Vault — 70+ official exam portals
📄 PYQ Papers — verified PDF links for 17 major exams
📰 Live News — real Google News, no fake dates
📝 To-Do List — tasks with priorities, saves automatically
👥 Multi-user — each person has private data, API key, chats


API Key (Free)

Go to dashboard.cohere.com/api-keys
Sign up → New Trial Key → Copy
Paste in the red banner inside the app


Notes

All data saves locally in your browser — nothing sent to any server except AI calls
Use a real browser (not incognito) so data persists after closing
Each user's API key, chats, and todos are completely private
For exams not in the list — AI still helps using its own knowledge
