## 🚀 What Is This?

ExamIntel Pro is a single HTML file — no installation, no server, no internet required for core features. Just open it in any browser and start studying.

Built for Indian competitive exam aspirants. Covers **150+ exams** across engineering, medical, management, civil services, banking, defence, law, teaching, design, and more.

---

## ⚡ Quick Start

1. **Download** `ExamIntel-V2.html`
2. **Open** in Chrome / Edge / Firefox
3. **Create your account** (name + password)
4. **Fill your profile** (class, college, target exam)
5. **Get your Cohere API key** (free) → paste it in the red box
6. **Start using** — timer starts, AI activates, everything saves

---

## 🔑 Getting Your Free API Key (2 minutes)

1. Go to → **[dashboard.cohere.com/api-keys](https://dashboard.cohere.com/api-keys)**
2. Sign up with Google (free, no credit card)
3. Click **"New Trial Key"**
4. Copy the key (starts with letters and numbers)
5. Open the app → **Predictor tab** → paste in the **red banner**
6. Click **💾 Save**, then **🧪 Test Key** to verify

> ⚠️ Your API key is saved **per user profile**, never shared with other users.

## 👥 Multi-User System

Multiple people can use the same file — each with their own private account.

### Creating an Account
- Open the app → click **➕ Add User**
- Enter your **name** and **password** (min 4 characters)
- Fill your **class**, **college**, and **target exam** (required)
- Start using — your data is completely separate from others

### Switching Users
- Click the **dropdown** at the top right
- Select the user you want to switch to
- Enter **that user's password** to confirm
- Everything switches — API key, chats, to-dos, theme

### Each User Has Their Own:
| What | Description |
|------|-------------|
| 🔑 API Key | Private, never visible to other users |
| 💬 Chat History | Multiple named conversations |
| 📝 To-Do List | Study tasks with priorities |
| ⏱️ Exam Timer | Countdown to their specific exam |
| 🎨 Theme | Light, Dark, Purple, or Green |
| 📋 Profile | Class, college, exam, institution |

## 🗂️ Features

### ⏱️ Exam Countdown Timer
- Real-time countdown: **Days + Hours + Minutes + Seconds**
- Updates every second
- Covers **150+ exams** with scheduled dates
- Shows in the main banner bar on every tab
- Big bold timer on the **Predictor tab**
- Changes to red/urgent when under 30 days
- If your exam isn't listed, timer shows `--` (still fully functional app)

### 🔮 Paper Predictor
- Upload **2 or more previous year PDFs** of your exam
- AI analyzes patterns across all uploaded papers
- Generates:
  - 🎯 High probability topics with % likelihood
  - 📝 **12 actual predicted questions** (real question statements)
  - 📊 Chapter-wise marks weightage
  - 🔮 Difficulty split (easy/medium/hard)
  - ⚠️ Wildcard topics
  - 💡 Last-week focus areas
- Results open as a **flip-book** — questions first, answers at the end
- Also available as a **formatted full-page analysis**

### 📖 Book Viewer
- Predictions open as a realistic page-turning book
- **Click right** → next page | **Click left** → previous page
- Or use ← → buttons at the bottom
- Questions on Pages 1–N, Answers on final pages
- Smooth paper-thin page turn animation

### 🤖 AI Study Assistant (Sidebar)
- Ask **anything** — math, science, history, strategy, tips
- AI knows your **name, class, college, and exact exam**
- Knows full syllabus, paper pattern, books, cutoffs for your exam
- Knows **exact days to your exam** — no wrong time calculations
- Uses a **real calculator tool** for all math (no guessing)
- Multiple named chats — like Claude/ChatGPT conversation history
- **Chat history saves** and loads when you log back in
- Double-click any chat name to rename it
- ⚠️ Always verify math answers — AI can occasionally err on complex calculations

### 🔢 Scientific Calculator
- Looks and works like a real phone calculator
- Full number pad + operations
- Scientific functions: sin, cos, tan, log, √, xʸ, π, e, ( )
- **Same calculator the AI uses** — guaranteed accurate results
- Powered by **math.js**
- Examples: `-9+2` → `-7` | `sqrt(144)` → `12` | `sin(pi/2)` → `1`

### 📚 Exam Vault
- **70+ exams** organized by category
- Engineering / Medical / Management / Civil Services / Banking / SSC / Railways / Defence / Law / Integrated / PG / Design
- Click any card → opens the official exam portal
- "Show More" expands each category
- Searchable

### 📄 Previous Year Question Papers
- **17 major exams** with verified paper links
- Sources: Official sites, Careers360, Embibe, Vedantu, Adda247, Oliveboard
- All links verified to go to actual PDF paper pages
- Years covered: typically 2018–2024/2025

### 📰 Live Exam News
- Fetches **real news** from Google News RSS
- Searches your exam + JEE/NEET/UPSC/GATE mix
- Sorted newest first with real dates
- Click any card → opens the full article
- No API key needed for news
- Fallback links if internet is slow

### 📝 To-Do List
- Add study tasks with **priority levels**: 🔴 High / 🟡 Medium / 🟢 Low / Normal
- Filter by: All / Pending / Done
- Shows stats: total, pending, done
- Saves automatically, persists across sessions
- Each user has their own separate to-do list

### 💪 Motivation
- Rotating motivational quotes every 8 seconds
- Tied to your exam countdown
- Changes quote style when under 30 days

## 💾 Data & Privacy

- **All data stored locally** in your browser (localStorage)
- Nothing is sent to any server except your API calls to Cohere
- Each user's data is completely isolated
- Clearing browser data / incognito mode will delete saved data
- Back up important data manually if needed

## 🛠️ Troubleshooting

| Problem | Fix |
|---------|-----|
| AI says "check API key" | Click 🧪 Test Key in the red banner |
| News not loading | Check internet, click Refresh again |
| Timer shows `--` | Your exam might not be in the date list yet — timer shows `?` |
| Chats not saving | Make sure localStorage isn't blocked (not incognito) |
| Calculator shows Error | Check expression syntax — use `*` not `x`, `sqrt(` not `√(` |
| Can't find exam in dropdown | Just type the full name freely — it saves as-is |
